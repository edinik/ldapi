# 通用思考指南

这些指南是 LDAPI 跨层变更的检查清单，用于补充 frontend、backend、testing 和 operations 中的具体实现规范。

| 指南 | 阅读时机 |
|---|---|
| [代码复用思考指南](./code-reuse-thinking-guide.md) | 辅助函数、组件、解析器、查询、请求封装或配置值可能已有所有者时 |
| [跨层思考指南](./cross-layer-thinking-guide.md) | 字段或行为跨越 UI、请求、服务、数据库、投影、部署或测试时 |

## 强制触发条件

- 新增辅助函数或全局样式前，先搜索既有所有者。
- 修改字段、环境变量、迁移或序列化值前，先映射所有读取方和写入方。
- 接受 review finding 前，必须根据源码和真实信任边界验证。

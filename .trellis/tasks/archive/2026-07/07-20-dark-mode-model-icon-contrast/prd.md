# 优化夜间模式模型图标对比度

## Goal

提升首页模型卡片中开发者图标在夜间模式下的可辨识度，同时保持日间模式现有视觉效果。

## Background

- 首页模型卡片由 `src/components/ModelOverview.tsx` 渲染。
- `DeveloperIcon` 当前直接通过 `<img>` 显示开发者或模型配置的图片图标，未针对深色主题调整前景色。
- 黑色或深色的单色图标在夜间卡片背景上对比度不足。

## Requirements

- 夜间模式下，模型卡片中的深色单色开发者图标应显示为清晰的浅灰白色。
- 日间模式下图标外观保持不变。
- 无图片图标时的文字回退样式保持可辨识。
- 修改范围限定在首页模型卡片的开发者图标，不改变后台表单、数据结构或图标路径映射。
- `AGENTS.md` 在 Trellis 托管块之外记录仅限本地开发使用的默认登录凭据，并明确禁止用于生产或共享环境。

## Acceptance Criteria

- [x] OpenAI 等深色单色图标在夜间模式下与卡片背景具有明显对比度。
- [x] 日间模式中的图标颜色不受影响。
- [x] 模型卡片布局、尺寸及文字回退图标保持原有行为。
- [x] 前端 lint、类型检查或项目中对应的最小相关检查通过。
- [x] `AGENTS.md` 已记录本地默认用户名 `admin` 和密码 `changeme`，并包含用途限制。

## Out of Scope

- 重绘或替换开发者图标资源。
- 调整模型卡片的其他颜色、排版或间距。
- 修改站点、模型或资源管理功能。
